# mlops-unit1

## Project Objective
This project demonstrates reproducibility using Python and Git in a simple machine learning workflow.

## Dataset Used
The Iris dataset from scikit-learn is used for classification.

## Project Structure
mlops-unit1/
├── data/
├── src/
├── models/
├── requirements.txt
└── README.md

## Steps to Run
1. Create a virtual environment
2. Activate the environment
3. Install dependencies using requirements.txt
4. Run the Python script

## Commands
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python src/main.py
